Python venv setup for MLDPPCC

1. Create a virtual environment

   python3.11 -m venv .venv

   If python3.11 is not available, use python3 instead.

2. Activate the environment

   Linux or macOS:
   source .venv/bin/activate

   Windows PowerShell:
   .venv\Scripts\Activate.ps1

3. Upgrade pip and install the project requirements

   python -m pip install --upgrade pip
   pip install -r requirements.txt

4. Register the environment as a Jupyter kernel

   python -m ipykernel install --user --name mldppcc

5. Prepare the data files

   Put photon.csv and neutron.csv inside Raw_Labelled_Waveforms/

6. Generate the cached datasets

   python preprocess.py Raw_Labelled_Waveforms processed_waveforms.npz
   python generate_pileup.py processed_waveforms.npz pileup_waveforms.npz

7. Open Final_Notebook_MLDPPCC.ipynb and select the mldppcc kernel

Notes

- If figure export fails because LaTeX is not installed, set plt.rcParams["text.usetex"] = False in the plotting cell
- The notebook expects Raw_Labelled_Waveforms/ to be in the same directory as the notebook and scripts
